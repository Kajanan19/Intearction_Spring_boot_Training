What is JavaScript?

JavaScript (JS) is a lightweight, interpreted, or 
just-in-time compiled programming language with first-class functions.
While it is most well-known as the scripting language for Web pages, 
many non-browser environments also use it

lightweight-: Small Memory foot print, easy to implement.
interpreted-: No Compilation, Instruction executed directly
object oriented-: Modeled with objects
first class functions-: functions can be a Value.
scripting language-: Instruction written for an run time environments.
